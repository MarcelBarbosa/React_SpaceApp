## Descrição:
Projeto utilizando React e Vite para criação de uma página de galerias que permite favoritar, expandir, buscar por tags e pesquisar textos em um campo de busca.


## Principais Ferramentas:
* Javascript
* React
* Styled Components
* Hooks
* Vite
